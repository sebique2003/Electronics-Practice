# Diode

## EN

A diode allows current to flow in only one direction.

If reversed, the current is blocked.

Uses:
- rectifiers
- protection
- LEDs


## RO

O diodă permite curentului să circule într-o singură direcție.

Dacă este inversată, curentul este blocat.

Utilizări:
- redresoare
- protecție
- LED-uri
