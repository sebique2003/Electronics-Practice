# PCB (Printed Circuit Board)

## EN

A permanent board that connects components using copper tracks.

**Key Terms:**
- **Trace:** A copper "wire" on the board.
- **Pad:** The spot where a component is soldered.
- **Soldering:** The process of joining components using melted metal (solder).

## RO

O placă permanentă care conectează componentele folosind trasee de cupru.

**Termeni Cheie:**
- **Traseu (Trace):** Un "fir" de cupru pe placă.
- **Pad:** Locul unde se lipește componenta.
- **Lipire (Soldering):** Procesul de îmbinare a componentelor folosind metal topit (cositor).
